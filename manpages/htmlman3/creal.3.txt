Name
creal, crealf, creall — get real part of a complex
      number

Synopsis

#include <complex.h>

double
            creal(
double complex z);

           
        

float
            crealf(
float complex z);

           
        

long double
            creall(
long double complex z);

           
        

Note

Link with −lm.

 
DESCRIPTION
The creal() function returns
      the real part of the complex number z.
One has:

    z = creal(z) + I * cimag(z)

 
VERSIONS
These functions first appeared in glibc in version
      2.1.

 
CONFORMING TO
C99.

 
NOTES
The gcc supports also __real__. That is a GNU
      extension.

 
SEE ALSO
cabs(3), cimag(3), complex(7)
