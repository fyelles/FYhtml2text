Section 3
 

 
 

Name
rewinddir — reset directory stream

Synopsis

#include <sys/types.h>

#include <dirent.h>

void
            rewinddir(
DIR *dirp);

           
        

 
DESCRIPTION
The rewinddir() function
      resets the position of the directory stream dirp to the beginning of the
      directory.

 
RETURN VALUE
The rewinddir() function
      returns no value.

 
CONFORMING TO
SVr4, 4.3BSD, POSIX.1-2001.

 
SEE ALSO
closedir(3), opendir(3), readdir(3), scandir(3), seekdir(3), telldir(3)
