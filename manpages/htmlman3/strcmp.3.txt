Section 3
 

 
 

Name
strcmp, strncmp — compare two strings

Synopsis

#include <string.h>

int
            strcmp(
const char *s1,

 
const char *s2);

           
        

int
            strncmp(
const char *s1,

 
const char *s2,

 
size_t n);

           
        

 
DESCRIPTION
The strcmp() function
      compares the two strings s1 and s2. It returns an integer less
      than, equal to, or greater than zero if s1 is found, respectively, to
      be less than, to match, or be greater than s2.
The strncmp() function is
      similar, except it only compares the first (at most)
      n characters of
      s1 and s2.

 
RETURN VALUE
The strcmp() and
      strncmp() functions return an
      integer less than, equal to, or greater than zero if
      s1 (or the first
      n bytes thereof) is
      found, respectively, to be less than, to match, or be greater
      than s2.

 
CONFORMING TO
SVr4, 4.3BSD, C89, C99.

 
SEE ALSO
bcmp(3), memcmp(3), strcasecmp(3), strcoll(3), string(3), strncasecmp(3), strverscmp(3), wcscmp(3), wcsncmp(3)
