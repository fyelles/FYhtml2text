Name
ctan, ctanf, ctanl — complex tangent function

Synopsis

#include <complex.h>

double complex
            ctan(
double complex z);

           
        

float complex
            ctanf(
float complex z);

           
        

long double complex
            ctanl(
long double complex z);

           
        

Note

Link with −lm.

 
DESCRIPTION
The complex tangent function is defined as:

    ctan(z) = csin(z) / ccos(z)

 
VERSIONS
These functions first appeared in glibc in version
      2.1.

 
CONFORMING TO
C99.

 
SEE ALSO
cabs(3), catan(3), ccos(3), csin(3), complex(7)
