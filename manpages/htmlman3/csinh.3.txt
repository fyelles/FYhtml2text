Name
csinh, csinhf, csinhl — complex hyperbolic sine

Synopsis

#include <complex.h>

double complex
            csinh(
double complex z);

           
        

float complex
            csinhf(
float complex z);

           
        

long double complex
            csinhl(
long double complex z);

           
        

Note

Link with −lm.

 
DESCRIPTION
The complex hyperbolic sine function is defined as:

    csinh(z) = (exp(z)−exp(−z))/2

 
VERSIONS
These functions first appeared in glibc in version
      2.1.

 
CONFORMING TO
C99.

 
SEE ALSO
cabs(3), casinh(3), ccosh(3), ctanh(3), complex(7)
