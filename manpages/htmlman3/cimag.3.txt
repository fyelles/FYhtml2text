Name
cimag, cimagf, cimagl — get imaginary part of a
      complex number

Synopsis

#include <complex.h>

double
            cimag(
double complex z);

           
        

float
            cimagf(
float complex z);

           
        

long double
            cimagl(
long double complex z);

           
        

Note

Link with −lm.

 
DESCRIPTION
The cimag() function returns
      the imaginary part of the complex number z.
One has:

    z = creal(z) + I * cimag(z)

 
VERSIONS
These functions first appeared in glibc in version
      2.1.

 
CONFORMING TO
C99.

 
NOTES
gcc also supports __imag__. That is a GNU extension.

 
SEE ALSO
cabs(3), creal(3), complex(7)
