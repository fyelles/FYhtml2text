Section 3
 

 
 

Name
seekdir — set the position of the next readdir()
      call in the directory stream.

Synopsis

#include <dirent.h>

void
            seekdir(
DIR *dirp,

 
long offset);

           
        

Note

Feature Test Macro Requirements
                      for glibc (see feature_test_macros(7)):

seekdir():

_BSD_SOURCE ||
                      _SVID_SOURCE ||
                      _XOPEN_SOURCE

 
DESCRIPTION
The seekdir() function sets
      the location in the directory stream from which the next
      readdir(2) call will start.
      seekdir() should be used with
      an offset returned by
      telldir(3).

 
RETURN VALUE
The seekdir() function
      returns no value.

 
CONFORMING TO
4.3BSD, POSIX.1-2001.

 
NOTES
In glibc up to version 2.1.1, the type of the offset argument was
      off_t. POSIX.1-2001 specifies
      long, and this is the type used
      since glibc 2.1.2.

 
SEE ALSO
lseek(2), closedir(3), opendir(3), readdir(3), rewinddir(3), scandir(3), telldir(3)
