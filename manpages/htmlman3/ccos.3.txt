Name
ccos, ccosf, ccosl — complex cosine function

Synopsis

#include <complex.h>

double complex
            ccos(
double complex z);

           
        

float complex
            ccosf(
float complex z);

           
        

long double complex
            ccosl(
long double complex z);

           
        

Note

Link with −lm.

 
DESCRIPTION
The complex cosine function is defined as:

    ccos(z) = (exp(i * z) + exp(−i * z)) / 2

 
VERSIONS
These functions first appeared in glibc in version
      2.1.

 
CONFORMING TO
C99.

 
SEE ALSO
cabs(3), cacos(3), csin(3), ctan(3), complex(7)
