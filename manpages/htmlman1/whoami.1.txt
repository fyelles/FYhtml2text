Section 1
 

 
 

Name
whoami — print effective userid

Synopsis

whoami [OPTION...]

 
DESCRIPTION
Print the user name associated with the current effective
      user ID. Same as id −un.

−−help

display this help and exit

−−version

output version information and exit

 
AUTHOR
Written by Richard Mlynarik.
