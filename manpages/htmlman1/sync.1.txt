Section 1
 

 
 

Name
sync — flush file system buffers

Synopsis

sync [OPTION]

 
DESCRIPTION
Force changed blocks to disk, update the super block.

−−help

display this help and exit

−−version

output version information and exit

 
AUTHOR
Written by Jim Meyering.
