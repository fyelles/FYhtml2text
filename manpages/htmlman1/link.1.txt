Section 1
 

 
 

Name
link — call the link function to create a link to a
      file

Synopsis

link FILE1 FILE2

link OPTION

 
DESCRIPTION
Call the link function to create a link named FILE2 to an
      existing FILE1.

−−help

display this help and exit

−−version

output version information and exit
